================================================================================
DARK FOREST — COMPREHENSIVE SOUND EFFECTS LIST
================================================================================

Legend:
  Type:      ONE-SHOT  = fire-and-forget, plays once
             LOOP      = continuous, starts/stops with state
             RAMP      = crossfades or pitch-shifts based on a parameter
             SEQUENCE  = multiple sounds played in timed order
  Priority:  CRITICAL  = core gameplay feel, needed first
             HIGH      = important feedback, noticeable if absent
             MEDIUM    = polish layer, adds texture
             LOW       = nice-to-have, ambient seasoning
  Spatial:   2D        = UI/cockpit, no positional falloff
             3D        = world-space, distance + direction attenuation


================================================================================
1. FLIGHT & PROPULSION
================================================================================

1.1  ENGINE IDLE HUM
     Type:     LOOP
     Spatial:  2D
     Priority: CRITICAL
     Trigger:  Always playing when ship is powered on
     Notes:    Low rumble/hum, very subtle. Provides baseline "you are in a ship"
               texture. Should feel like sitting in a submarine or cockpit.

1.2  THRUST INCREASE / ACCELERATION
     Type:     RAMP (pitch/volume scales with targetSpeed)
     Spatial:  2D
     Priority: CRITICAL
     Trigger:  Shift held, speed increasing
     Notes:    Layered over engine idle. Higher pitch and volume as speed climbs.
               Think jet turbine spool-up. Could crossfade between 2-3 intensity
               layers (low/med/high thrust).

1.3  THRUST DECREASE / DECELERATION
     Type:     RAMP (pitch/volume scales down with speed)
     Spatial:  2D
     Priority: CRITICAL
     Trigger:  Ctrl held, speed decreasing
     Notes:    Reverse of 1.2. Engine winding down, slight braking texture.

1.4  MWD ACTIVATION
     Type:     ONE-SHOT (leading into 1.5 loop)
     Spatial:  2D
     Priority: CRITICAL
     Trigger:  Enter key / MWD button pressed
     Notes:    Dramatic ignition burst. Deep bass punch + high-frequency crackle.
               Capacitor drain audible as power surge. ~0.5-1s.

1.5  MWD BURN (SUSTAINED)
     Type:     LOOP
     Spatial:  2D
     Priority: CRITICAL
     Trigger:  mwdActive === true (5-20s duration)
     Notes:    Roaring sustained burn. Should feel powerful and urgent. Deep
               rumbling with slight oscillation. Distinct from normal thrust.

1.6  MWD SHUTDOWN
     Type:     ONE-SHOT
     Spatial:  2D
     Priority: CRITICAL
     Trigger:  mwdRemaining reaches 0
     Notes:    Engine flame-out / power-down. Inverse of activation.
               Descending pitch, ~0.5s. Transitions back to normal engine.

1.7  MWD COOLDOWN TICK (OPTIONAL)
     Type:     ONE-SHOT (repeating)
     Spatial:  2D
     Priority: LOW
     Notes:    Subtle tick/pulse while MWD is on cooldown. Lets pilot know
               system is recovering without looking at the UI.

1.8  DAMPENERS ENGAGE
     Type:     ONE-SHOT
     Spatial:  2D
     Priority: HIGH
     Trigger:  Space key / DMP button — dampenersActive → true
     Notes:    Mechanical clunk + electromagnetic hum onset. "Systems coming
               online" feel. ~0.3s.

1.9  DAMPENERS DISENGAGE
     Type:     ONE-SHOT
     Spatial:  2D
     Priority: HIGH
     Trigger:  Space key / DMP button — dampenersActive → false
     Notes:    Power-down whine, mechanical release. Ship now drifting free.
               ~0.3s.

1.10 CAPACITOR DEPLETED WARNING
     Type:     ONE-SHOT
     Spatial:  2D
     Priority: HIGH
     Trigger:  Capacitor reaches 0, targetSpeed forced to 0
     Notes:    Alarm beep or power-failure stinger. Short, attention-grabbing.
               Distinct from combat warnings.

2. WARP DRIVE
================================================================================

2.1  WARP SPOOL-UP / ALIGNMENT
     Type:     RAMP (builds over alignment period)
     Spatial:  2D
     Priority: CRITICAL
     Trigger:  warpState → 'aligning'
     Notes:    Rising energy buildup. Oscillating hum that increases in pitch
               and intensity. Capacitor drain audible. ~2-4s depending on
               alignment time.

2.2  WARP DEPARTURE (GRID EXIT)
     Type:     ONE-SHOT + TRANSITION
     Spatial:  2D
     Priority: CRITICAL
     Trigger:  warpState → 'warping', SOURCE_GRID_EXIT_DURATION_MS (1.5s)
     Notes:    Climactic burst as ship enters warp. Bass drop + high-frequency
               stretch effect. The "jump" moment. Transitions into 2.3.

2.3  WARP CRUISE
     Type:     LOOP
     Spatial:  2D
     Priority: CRITICAL
     Trigger:  In warp transit (distance-dependent duration)
     Notes:    Sustained warp tunnel sound. Deep, resonant, slightly unsettling
               hum. Muffled quality — everything else feels far away. Could
               have subtle pitch variation over time.

2.4  WARP DECELERATION / LANDING
     Type:     RAMP (descending over 4.2s)
     Spatial:  2D
     Priority: CRITICAL
     Trigger:  warpState → 'landing', LANDING_DURATION_MS (4.2s)
     Notes:    Warp sound winding down, reality reasserting. Descending pitch,
               increasing clarity of normal ship sounds. Ends with a thud/boom.

2.5  WARP COMPLETE
     Type:     ONE-SHOT
     Spatial:  2D
     Priority: HIGH
     Trigger:  warpState → 'idle' after landing
     Notes:    Final arrival punctuation. Short bass thump + systems
               normalizing. ~0.5s.

3. COMBAT — ORDNANCE
================================================================================

3.1  TORPEDO LAUNCH
     Type:     ONE-SHOT
     Spatial:  2D (own ship), 3D (other ships)
     Priority: CRITICAL
     Trigger:  LAUNCH button, launchLockedCylinder()
     Notes:    Mechanical tube release + propulsion ignition. Concussive thump
               felt through the hull, then a fading whoosh as torpedo departs.
               ~1s.

3.2  TORPEDO IN FLIGHT (NEARBY)
     Type:     LOOP (3D positioned, Doppler)
     Spatial:  3D
     Priority: MEDIUM
     Trigger:  Torpedo within audible range (~2000m)
     Notes:    High-pitched whine/hiss of torpedo motor. Should Doppler shift
               as it passes. Fades with distance. Only heard if torpedo is
               close to player ship.

3.3  TORPEDO IMPACT / EXPLOSION
     Type:     ONE-SHOT
     Spatial:  3D (distant), 2D (when hitting own ship)
     Priority: CRITICAL
     Trigger:  Torpedo reaches target within TORPEDO_HIT_RADIUS (50m)
     Notes:    Two variants needed:
               a) DISTANT: Muffled explosion, bass-heavy. Attenuates with
                  distance.
               b) OWN SHIP HIT: Full-body impact. Screen-shaking bass + metal
                  stress sounds. Layered with shield/armor/hull impact (3.6-3.8).
               Explosion visual lasts 7.2s but sound should be ~2-3s.

3.4  FLARE LAUNCH
     Type:     ONE-SHOT
     Spatial:  2D
     Priority: HIGH
     Trigger:  FLARE button, launchFlares()
     Notes:    Burst of 5 flares at 1680 m/s. Pop-pop-pop rapid ejection
               sound, like pyrotechnic charges. ~0.5s.

3.5  FLARE BURN (NEARBY)
     Type:     LOOP (short, 3s lifetime)
     Spatial:  3D
     Priority: LOW
     Trigger:  Flares in flight
     Notes:    Crackling/hissing of burning flares. Fades as they disperse.

4. COMBAT — DAMAGE & DEFENSE
================================================================================

4.1  SHIELD ACTIVATION
     Type:     ONE-SHOT + RAMP (online level ramps over ~2s)
     Spatial:  2D
     Priority: CRITICAL
     Trigger:  SHD button → shieldsUp = true
     Notes:    Electromagnetic power-up. Rising hum that stabilizes as
               shieldOnlineLevel reaches full. Energy field engaging.

4.2  SHIELD DEACTIVATION
     Type:     ONE-SHOT
     Spatial:  2D
     Priority: HIGH
     Trigger:  SHD button → shieldsUp = false
     Notes:    Power-down, shield field collapsing. Descending whine. ~0.5s.

4.3  SHIELD AMBIENT HUM
     Type:     LOOP
     Spatial:  2D
     Priority: MEDIUM
     Trigger:  While shields are online
     Notes:    Very subtle background hum indicating shields are up. Almost
               subliminal. Mixed low under engine sounds.

4.4  SHIELD RECHARGE
     Type:     LOOP (subtle)
     Spatial:  2D
     Priority: LOW
     Trigger:  Shield recharging (capacitor draining to shields)
     Notes:    Quiet electrical pulsing. Indicates cap-to-shield transfer.
               Intensity could scale with recharge rate (0-100%).

4.5  SHIELD IMPACT
     Type:     ONE-SHOT
     Spatial:  2D
     Priority: CRITICAL
     Trigger:  Torpedo hits while shields absorb damage
     Notes:    Energy dispersal crackle. Electric/plasma discharge sound.
               Shield absorbing the blow. Should feel protective but alarming.

4.6  ARMOR IMPACT
     Type:     ONE-SHOT
     Spatial:  2D
     Priority: CRITICAL
     Trigger:  Torpedo hits armor (shields depleted or bypassed)
     Notes:    Metallic crunch/grinding. Physical impact on hull plating.
               More visceral and concerning than shield hit.

4.7  HULL IMPACT
     Type:     ONE-SHOT
     Spatial:  2D
     Priority: CRITICAL
     Trigger:  Torpedo hits hull (armor depleted)
     Notes:    Deep structural groan + rupture. Most alarming damage sound.
               Metal tearing, pressurization alarms. Player should feel dread.

4.8  DAMAGE LAYER TRANSITION (OPTIONAL)
     Type:     ONE-SHOT
     Spatial:  2D
     Priority: HIGH
     Trigger:  Shield → Armor or Armor → Hull transition (layer fully depleted)
     Notes:    Warning stinger indicating defensive layer is gone.
               "Shields down!" moment.
