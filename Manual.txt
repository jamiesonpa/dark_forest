Dark Forest - Player Manual
Version: Living document (updated with gameplay changes)
Last Updated: 2026-03-19

Introduction
Dark Forest is a multicrew ship combat simulator game. Crew members can switch stations at any time, but the best results come from coordinated roles:
- Pilot controls ship motion, attitude, warp entry, and defensive systems.
- EWO (Electronic Warfare Officer) manages sensors, contact tracking, lock management, and target data for the team.
- WSO (Weapons and Systems Officer) manages IRST-based targeting, torpedoes, directed-energy firing workflow, and flare deployment controls.

This manual documents controls and workflows that are currently implemented in the game build.

========================================
Global Controls (All Stations)
========================================

Station Switching
- F1 -> Pilot station
- F2 -> EWO station
- F3 -> WSO station

Ship-Wide Quick Toggles
- Backspace -> Toggle nav attitude mode (AA <-> DAC)
  - Only triggers when you are not typing in an input field.

- Enter -> Activate MWD
  - Only triggers when you are not typing in an input field.
  - If MWD cannot activate (cooldown/capacitor/other gating), the request is ignored.

========================================
Role: Pilot
========================================

Pilot Responsibilities
- Fly and orient the ship.
- Manage speed, dampeners, shields, MWD timing, and warp execution.
- Keep the ship aligned with EW/WS targeting and mission objectives.

Pilot Keyboard Flight Controls
- A / D -> Yaw command (bearing left/right)
- W / S -> Pitch command (inclination up/down)
- Shift (left or right) -> Increase target speed
- Ctrl (left or right) -> Decrease target speed
- Space -> Toggle inertial dampeners on/off
- G -> Engage warp to selected destination (only if alignment and capacitor checks pass)
- Q / E -> Roll control in DAC mode

Pilot HUD / Mouse Controls
- Nav Mode toggle:
  - AA = Attitude Assist
  - DAC = Direct Attitude Control
- Nav solution panel (AA + dampeners online):
  - Click/point in bearing scope to set commanded bearing.
  - Click/point in inclination meter to set commanded inclination.
- DMP button -> Toggle dampeners.
- SHD button -> Raise/lower shields.
- MWD panel:
  - Set MWD duration with the slider.
  - Press MWD button to start burn when available.
- Warp panel:
  - Set warp arrival distance with slider.
  - Press WARP button when ready.

Pilot Operating Notes
- In AA mode, ship tracks commanded attitude.
- In DAC mode, controls become direct attitude inputs and Q/E roll is active.
- With dampeners offline, movement/inertia behavior changes significantly; coordinate this with EW before aggressive maneuvers.

========================================
Role: EWO (Electronic Warfare Officer)
========================================

EWO Responsibilities
- Build and maintain tactical awareness.
- Detect and classify contacts.
- Manage lock states and feed target quality to the ship/crew.
- Control scanner configuration and emissions discipline.

Primary EWO Console Areas
- Scanner A and Scanner B:
  - Each scanner can be set to GRAV or IR.
  - Each scanner can be powered ON/OFF.
  - RESET clears scanner gate state.
- Multi-Function Display tabs:
  - MAP tab: 3D tactical map interaction.
  - RADAR tab: B-scope style radar picture and lock management.

EWO Scanner Controls (Mouse)
- GRAV/IR analyzer canvases:
  - Left drag to set gate region.
  - Mouse wheel to zoom gate width.
  - Right click to clear gate.
- GRAV scanner:
  - Adjust GAIN and HPF sliders.
  - ANALYZE button starts gravimetric analysis when a valid in-gate target is available.
- IR scanner:
  - Adjust GAIN slider.
  - IRST SEND pushes the gated thermal target to ship IRST bearing/inclination when valid.

EWO Map Controls (MAP tab)
- Left drag -> Orbit map camera.
- Mouse wheel -> Zoom.
- Click marker -> Select track/celestial marker.

EWO Radar Controls (RADAR tab)
- Range controls:
  - Use + / - buttons to change B-scope range scale.
  - Mouse wheel over radar scope also adjusts range.
- Contact locking:
  - Right click a radar track -> Set HARD lock on that track.
  - Right click empty radar scope -> Clear hard lock(s).
- Bearing display mode:
  - REL and ABS toggle changes bearing reference readout.

EWO Operating Notes
- During warp transit, radar page is inoperable.
- Radar emissions increase observability (EMCON warning in header/status).
- Scanner power state and mode selection directly affect available sensor workflows.

========================================
Role: WSO (Weapons and Systems Officer)
========================================

WSO Responsibilities
- Manage IRST camera setup and target tracking.
- Load and launch torpedo tubes.
- Operate directed energy weapon charging and fire timing.
- Deploy countermeasures (flares) and monitor stores.

WSO Console Areas
- IRST Targeting panel (left).
- Torpedo Tubes and Torpedo Reserves.
- Directed Energy Weapon panel.
- Countermeasures panel.

WSO IRST Controls
- IRST power can be toggled directly on the IRST display.
- FWD button (hold) -> Slews IRST view back toward forward reference.
- BHOT/WHOT button -> Toggles thermal polarity (available in IR spectrum mode).
- IR/VIS button -> Toggles IRST spectrum mode.
- STAB button -> Toggles stabilized tracking frame.
- PT TRK button -> Attempts point track lock on hostile target near IRST boresight.
  - Requires target within off-boresight and range constraints.

WSO Torpedo Controls
- Four tubes are available.
- For each tube:
  - Select type while tube is EMPTY (IR SEEKER, ACTIVE, SEMI ACTIVE).
  - LOAD starts loading cycle.
  - LAUNCH fires when tube is READY and at least one EW soft/hard lock exists.
- Tube loading consumes reserve inventory for selected type.
- Torpedo Reserves panel shows remaining rounds by type.

WSO Directed Energy Weapon (DEW)
- PWR ON/OFF controls panel power.
- ARM/ARMED controls firing safety state.
- CHARGE begins charge cycle; FIRE becomes available once charged and target is valid.
- NO TGT indicates charge complete but no valid point-track target.
- RST cancels/reset charge state.
- Lens diagram supports drag-adjust focus alignment.
  - Alt + mouse wheel over a lens adjusts its width (curvature / effective focal length).
  - Focus % combines axial alignment (FOC vs target range) with beam spot size: vertical separation of the two ray traces where they meet the target range line (tighter = better).
  - Better focus increases applied DEW damage.

WSO Countermeasures
- PWR ON/OFF controls countermeasures subsystem.
- ARM/ARMED controls launch safety state.
- FLARES selector chooses launch count (1-8, limited by remaining inventory).
- PTN mode -> pattern launch.
- SGL mode -> queued single launches at short interval.
- FLR button executes flare launch.
- CHAFF status is currently displayed as readout; active chaff deployment control is not yet implemented.

========================================
Crew Coordination Checklist
========================================

Startup Flow
1) Pilot confirms dampener/nav mode baseline and propulsion readiness.
2) EWO powers/configures scanners and establishes initial contact picture.
3) WSO powers IRST/weapon subsystems and verifies point-track + tube/flare readiness.
4) Crew agrees on maneuver profile (AA vs DAC), EMCON posture, and initial engagement plan.

Combat / Contact Flow
1) EWO identifies and locks contacts.
2) WSO acquires PT track and prepares torpedo/DEW/countermeasure states.
3) Pilot keeps geometry favorable for sensors and weapon employment.
4) WSO executes torpedo, DEW, and flare timing based on lock quality and threat updates.

========================================
Quick Start (First 60 Seconds)
========================================

1) Press F1 (Pilot) and verify dampeners/nav mode baseline.
2) Press F2 (EWO), power scanners, and establish initial lock picture.
3) Press F3 (WSO), power IRST, arm relevant weapon systems, and confirm flare count/mode.
4) Coordinate heading/speed and EMCON posture between Pilot, EWO, and WSO.
5) Use Enter (MWD) and G/WARP only when capacitor/alignment conditions are met; engage weapons only once lock/track quality is confirmed.
